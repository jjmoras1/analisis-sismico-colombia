# Análisis de Sismicidad en Colombia (2020-2026)

## Descripción
Análisis exploratorio de 623 eventos sísmicos registrados en Colombia 
entre 2020 y 2026, usando el catálogo del USGS.

## Hallazgos principales
1. Colombia concentra el 61% de la sismicidad regional
2. Dos regímenes tectónicos diferenciados: subducción costera y Wadati-Benioff andino
3. Sin estacionalidad mensual ni tendencia anual significativa
4. Zonas sismogénicas identificadas: Bucaramanga, Cauca-Romeral, Chocó-Panamá y Piedemonte llanero

## Datos
Fuente: USGS Earthquake Catalog  
https://earthquake.usgs.gov/earthquakes/search/

## Tecnologías
Python, Pandas, Matplotlib, GeoPandas, Contextily, Folium
